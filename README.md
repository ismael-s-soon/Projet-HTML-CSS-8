## Style-Guide

**Palette de Couleurs**
- Primary : #2563eb, Primary Dark : #1d4ed8  
- Secondary : #64748b, Success : #10b981  
- Warning : #f59e0b, Danger : #ef4444  
- Background : #f8fafc, Sidebar : #1e293b, White : #ffffff  

**Typographie**
- Police : 'Inter', taille de base 16px, line-height 1.6  
- Titres : H1 2rem → H6 1rem  
- Texte : Large 1.25rem, Base 1rem, Small 0.875rem  

**Espacement**
- Padding : 1rem cartes / 1.5rem sections  
- Margin : 1rem entre éléments / 2rem sections  
- Gap Grid : 1.5rem  

**Composants**
- Boutons : `.btn-primary`, `.btn-secondary`, `.btn-danger`  
- Cartes : `.card`, `.stat-card`  
- Badges : `.status-badge`  
- Formulaires : `.form-input`  

**Ombres**
- Shadow : léger à large pour cartes et boutons

**Responsive**
- Mobile < 768px, Tablet 768–1024px, Desktop > 1024px  
- Grilles adaptatives `.grid-4`  

**Animations**
- Transitions : 0.2s à 0.5s  
- Animations : fadeIn, slideUp, pulse  

**Graphiques**
- Couleurs : bleu, vert, orange, rouge, violet  
- Barres de progression `.progress-bar`, `.progress-fill`  

**Utilitaires**
- `.text-center`, `.mb-1`, `.p-1`, `.hidden`, `.rounded`
